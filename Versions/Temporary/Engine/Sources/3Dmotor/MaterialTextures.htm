Материалы и текстуры

Понятие текстуры разделено на два:

  • Material – задаёт свойства текстуры, такие, как отражение, отбрасывание тени, прозрачность и т.д.
  • Texture – задаёт параметры конвертации нарисованного художником tga файла в понятный игре формат.

Texture

  • SrcName Путь до tga файла с текстурой. Все пути задаются относительно R:\ Complete.
  • Type
    • Ordinary Обычная текстура
    • 2D Используется только для интерфейсных текстур
  • ConversionType
    • Convert_Ordinary Конвертировать «как есть», без альфы
    • Convert_Bump Не поддерживается
    • Convert_Transparent Накладываеть по с альфе
    • Convert_Transparent_Add             Конвертировать по add
    • Convert_Linear_Picture
  • AddrType
    • WRAP – текстура “ врапается” – при текстурных координатах больше 1 и меньше 0 идет повторение текстуры
    • CLAMP – текстура “ клампается” - при текстурных координатах больше 1 и меньше 0 идет повторение краевого значения цвета текстуры
    • Если вы не понимаете, что означают предыдущие два пункта, используйте CLAMP
  • Format
    • TF_DXT1 Текстура без альфы
    • TF_DXT3 Текстура с альфой
    • TF_8888 2В текстуры и particles
    • TF_565
  • FlipY      Исползьуется для particles

Остальные поля нужно оставлять по умолчанию.

Material

  • Texture ссылка на Texture
  • Bump вторая текстура
  • MetalMirror – степень зеркальности, от 0 до 1, при 1 объект ведет себя как полностью зеркальный i.e. металлический шар. В случае 100% отражения (например, MetailMirror установлен в 1) поверх обычного вида объекта будет нарисована текстура отражения. Текстура отражения берется из свойств ambient light`a и представляет собой cube map текстуру. Для каждой вершины объекта будет рассчитан вектор отражения луча света из камеры в этой вершине и эти вектора будут использованы для наложения cube map текстуры на объект (то есть отражение зависит от нормалей в вершинах). Если в альфа канале cube map текстуры будет не единичное значение, то cube map текстура будет наложена с учетом этого альфа канала.
  • DielMirror – тоже, что и MetalMirror, но степень зеркальности дополнительно зависит от угла под которым наблюдается объект, такими свойствами обладает, например, вода (и другие диэлектрики). Может использоваться совместно с MetalMirror. Степень зеркальности при совместном использовании будет равна сумме этих двух значений (то есть материал на 0.5 может вести себя как метал и на 0.5 как вода).
  • Mirror – текстура, на которую домножается цвет cube map текстуры отражения. Альфа канал текстуры домножается на альфа канал cube map текстуры и учитывается при наложении cube map текстуры. Т.е. текстура (32 бита) в цвете (градации серого) должна содержать степень отражения (белый - максимальное отражение, черный - отражения нет), а в альфа канале однобитную маску, вырезающую отражающую часть текстуры.
  • CastShadow – отбрасывает ли объект с таким материалом тень, некоторые типы материалов не могут отбрасывать тень (например, a_transparent не могут отбрасывать тень)
  • Priority – приоритет материала, влияет на порядок отрисовки overlay материалов, рекомендуется устанавливать в 0 и использовать в исключительных случаях
  • AlphaMode – поле используется для M_GENERIC и определяет режим прозрачности
    • AM_OPAQUE - обычный материал
    • AM_ALPHA_TEST - обычный материал с проверкой альфы (подходит для листвы деревьев и других случаев, когда нужны дырки в объектах)
    • AM_TRANSPARENT - прозрачные материал по альфе (отличается от AM_OVERLAY тем, что для них производится сортировка)
    • AM_OVERLAY - накладываемые по альфе разные дополнения ( декалы) (не сортируются, обычно правильно работают, если рисуются рядом с обычной геометрией, например, дырки от пуль)
  • Effect
    • M_GENERIC - обычный материал
    • M_WATER - материал воды b2
    • M_TRACKS - материал следов от танков итп
    • M_CLOUDS_ H5 – материал для облаков h5
    • M_ANIM_ WATER – материал воды h5
    • M_SURF – тоже про воду из h5
  • Is2Sided – 2 сторонний материал (работает для M_GENERIC)
  • LightingMode – режим освещения (работает для M_ GENERIC)
    • L_NORMAL – обычное освещение
    • L_SELFILLUM - материал не учитывает освещение
