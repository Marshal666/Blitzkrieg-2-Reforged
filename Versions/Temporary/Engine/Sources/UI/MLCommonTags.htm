Список общих тегов,
поддерживаемых ML-текстом
(interface IML)

Форматирование:
<br> - перевод строки
<tab> - табуляция
Выравнивание текста:
<left> - выравнивание по левому краю
<right> - выравнивание по правому краю
<center> - выравнивание по центру
<nowrap> - отключает автоматическое выравнивание
<justify> - выравнивание по обоим краям, с растяжением пробелов (может и не работать)
<wrapleft> - выравнивание по левому краю, с "откусыванием" пространства
<wrapright> - тоже, но по правому краю
<top> - выравнивание по верхнему краю (нужно, если в строке есть шрифты разного размера)
<bottom> - тоже, но по нижнему
<middle> - тоже, но по центру
Шрифт:
<font ...>
	size - размер шрифта (pt - в точках, px - в пикселах. Использовать только pt, тк он не зависит от разрешения!)
	face - название шрифта (из базы)	
	outlinesize - размер обводки
	outlinecolor - цвет обводки
	forcefontsize - заставляет игнорировать прибавку, создаваемую Outline-ом
Примеры: <font face=Courierbold outlinesize=1 outlinecolor=0xFF222012 forcefontsize=true size=27>
<minfontsize> - ?
Цвет:
<color ...> - цвет шрифта
	AARRGGBB - код цвета в шестнацетеричном формате
	Название цвета:
	white
	red
	green
	blue
	yellow
	cyan
	orange
	pink
	brown
	grey
	black
	darkyellow
	beige
	lightblue
Примеры: <color=FFCECFCE>
Картинка:
<image ...> - картинка
	align - left или right (это wrap-allign)
	id - номер картинки в базе UITexture
	width - ширина (если не задана, то берётся из картинки)
	height - высота (аналогично)
	border - отступы вокруг картинки
Сохранение/восстановление состояния:
<push> - сохранить текущее состояние (фонт, цвет и т.д.)
<pop> - восстановить сохраненное состояние
