Звук, добавленный в звуковую в SoundScene с максимальным радиусом (R) AI тайлов и wav-file длиной (L) bytes будет занимать:

1) PHS map: 16 *  (R / 4) * (R / 4) bytes

2) создастся запись в звуковой сцене: ~200 bytes

Звук, отправленный на проигрышь

1) Файл будет считан в память: L bytes

Статическая память

1) всякие классы: 1K

2) Звуковые клетки:  30 * (mapTileSizeX/4) * (mapTileSizeY/4)

 

таким образом, например 10 взрывов с R = 60 Тайлов, L = 75K, будет занимать 

10 * 3600 + 75000 ~= 100К
