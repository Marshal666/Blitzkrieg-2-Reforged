Какую информацию увидит игрок при выборе этой миссии, какой бонус получит игрок по прохождению.

Обеспчить эвакуацию французского золота. Конвой с французским золотом движется по дороге вдоль берега к порту эвакуации. Его преследуют немецкая передавая развед группа. По всей видимости она уже навела на конвой танки находящиеся в непосредственной близости. Если не предпринять срочных мер конвой будет перехвачен. Необходимо высадить наши войска на побережье и прирыть конвой.

Скриптовые события.

Камера показывает начало движения конвоя, конвой движется по скрипту, затем показывает наши баржи. Через некоторое время из за края вылезжают немецкие броневички преследующие конвой. Еще через некоторое время вылетает немецкий развед самолет, потом наш разведсамолет.

Немецкие катера выходят из за края и нападают на наши баржи.

Конвой проходит 1/3 пути. Первая танковая группа немцев пытается его перехватить.

Конвой проходит 1/2 пути. Вторая группа катеров пытается атаковать.

Конвой проходит 2/3 пути. Вторая группа танков пытается перехватить.
