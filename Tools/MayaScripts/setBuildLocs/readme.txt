"setBuildLocs"

Команда приклеивает локаторы на ближайшие фейсы с определенной ориентацией локальных осей.
Нужна для быстрого позиционирования локаторов эффектов для домов проекта В2.
Скопируйте setBuildLocs.mll в папку Maya/bin/plugins и запустите плагин в плагин менеджере.
Выделите сначала локаторы, потом меш и выполните команду setBuildLocs

Тех поддержка - Мельков Е.
